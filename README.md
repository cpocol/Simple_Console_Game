# Simple_Console_Game
Start building your own 2D game (Snake, Pac-Man, Tetris, Mario etc) in console/command prompt/ASCII.

![Snapshot](/Preview.gif)

Use arrow keys to control the "player".

Prepared to be opened with:
- MS Visual Studio
- CodeBlocks
